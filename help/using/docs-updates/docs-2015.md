      <li id="li_0B524C6767274E1BAADEE0833DA00C84"> <p>Les expressions de plafonnement des fréquences incluent tous les utilisateurs dont le nombre de réalisations de caractéristiques est inférieur à une valeur souhaitée. </p> <p>Par exemple : </p> <p> <code> frequency([1000T]) &lt;= 5</code> </p> <p>Cette expression inclut tous les utilisateurs qui ont réalisé la caractéristique avec l’ID "1 000" moins de cinq fois, y compris ceux qui n’ont pas réalisé la caractéristique. </p> </li>
