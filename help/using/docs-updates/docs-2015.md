      <li id="li_0B524C6767274E1BAADEE0833DA00C84"> <p>Les expressions de plafonnement de fréquence incluent tous les utilisateurs dont le nombre de realizations de caractéristiques est inférieur à la valeur souhaitée. </p> <p>Par exemple : </p> <p> <code> fréquence ([1000 T]) &lt; = 5</code> </p> <p>Cette expression inclut tous les utilisateurs qui ont réalisé la caractéristique avec l'identifiant « 1000 » moins de cinq fois, y compris les utilisateurs qui n'ont pas réalisé la caractéristique. </p> </li>
